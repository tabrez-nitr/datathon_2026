# Crime Intelligence Platform

An AI-powered investigative platform that enables law enforcement agencies to interact with crime records using natural language while providing crime analytics, criminal network visualization, case summarization, and intelligent decision support.

## Features

- AI Chat Assistant
- Crime Analytics
- Criminal Network Visualization
- Similar Case Retrieval
- Crime Heatmaps
- AI Case Summaries
- Voice Support
- Explainable AI

## Tech Stack

Frontend:
- Next.js
- TypeScript
- TailwindCSS

Backend:
- FastAPI
- PostgreSQL
- SQLAlchemy

AI:
- LangGraph
- Gemini
- pgVector
